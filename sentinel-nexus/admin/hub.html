OpenClaw + Jobmaster Agency

One place for all links. Two systems: OpenClaw (gateway/chat) and Jobmaster Agency (landing, jobs, admin).

OpenClaw (gateway & chat)

Runs on port 18789. Start with: openclaw gateway run

  • OpenClaw dashboard (add ?token=YOUR_GATEWAY_TOKEN from ~/.openclaw/openclaw.json)
  • OpenClaw chat (fresh session)

Token is in ~/.openclaw/openclaw.json → gateway.auth.token. Use openclaw-launch.html and paste the token once to open the UI.

Jobmaster Agency (this server, port 3880)

Runs on port 3880. Start with: node sentinel-nexus/admin/server.js or ./sentinel-nexus/run-agency-24-7.sh

  • Landing page
  • Post a job
  • Track your job
  • Admin → Dashboard · Workers · Completed · Analysis · Brain · Report demo

If admin links show “not found”, make sure the admin server is running: OPENCLAW_STATE_DIR=~/.openclaw node sentinel-nexus/admin/server.js
